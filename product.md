# Product Specification: MLSuite

## Overview
MLSuite is a comprehensive machine learning workflow automation platform designed to simplify the process of data ingestion, model training, and deployment.

## Key Features
- **Automated ML Tuning**: Supports Random Forest, Decision Tree, and Linear/Logistic Regression with intelligent hyperparameter search.
- **Role-Based Access**: Specialized Admin and User portals for secure system management.
- **Identity Vault**: Centralized user control for administrators.
- **Premium Design**: "Digital Nebula" theme with a focus on visual excellence and user engagement.

## Technical Stack
- **Frontend**: Vite, React, TypeScript, Tailwind CSS.
- **Backend**: FastAPI, Python, Scikit-learn.
- **DevOps**: GitHub Actions, Docker, Supabase.
